Latest Project

Criss-Cross Game

I developed a criss-cross game (also known as Tic-Tac-Toe) using C++ programming language. This project involved creating a user-friendly console application where two players can compete against each other. The game features: A grid-based game board Input validation to ensure players enter valid moves A win-checking algorithm to determine the game outcome The ability to restart the game after each round This project helped me enhance my understanding of object-oriented programming, improve my coding skills in C++, and strengthen my problem-solving abilities.

Spotify Clone

I developed a Spotify clone using React.js, which replicates the core functionalities of the popular music streaming service. Music Playback: Integration with the Spotify API to fetch and play music tracks. Search Functionality: Real-time search for songs, artists, and albums. Playlists: Users can create, edit, and manage their playlists. This project allowed me to deepen my knowledge of React.js, state management with Redux, and API integration. It also provided valuable experience in building a full-fledged web application with a focus on performance and user experience.

Line Follower Bot

I built a line follower bot using Arduino, motor drivers, and IR sensors, programmed with C. This project involved designing and assembling a robotic vehicle that can autonomously follow a predefined path. This project enhanced my practical knowledge of electronics, sensor integration, and embedded programming. It provided hands-on experience with hardware-software interaction and real-time system control.
